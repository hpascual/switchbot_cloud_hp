# SwitchBot Cloud HP

Custom version of the native Home Assistant SwitchBot Cloud integration with faster polling intervals.

## Features

- Based on Home Assistant native SwitchBot Cloud integration
- Polling interval reduced from 600 seconds to 10 seconds
- Compatible with HACS

## Installation

1. Open HACS
2. Integrations
3. Custom repositories
4. Add this repository URL
5. Category: Integration
6. Install
7. Restart Home Assistant
